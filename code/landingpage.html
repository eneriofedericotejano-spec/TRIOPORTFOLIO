Projects

FurEver Care Veterinary Services Marketing and Promotion

Fur-Ever Care Veterinary Services Marketing and Promotion is a modern, user-friendly digital platform that lets clinic staff easily create, update, and manage promotions such as discounts, pet care packages, and seasonal offers through a simple dashboard. This boosts visibility, strengthens client connections, and encourages loyalty, using modern customer engagement and technology to ensure clinics succeed in a digital world.

TECH STACK
  • React
  • Node.js
  • MySql
  • Tailwind

Quezon City University Management Portal

A web-based University Management Portal designed to streamline and centralize academic and administrative processes at Quezon City University. The system integrates student information management, enrollment processing, grade viewing, announcements, and user account management into a single, unified platform. By providing real-time access to academic records, automated workflows, and an intuitive dashboard, the portal enhances operational efficiency, improves data accuracy, ensures transparency, and supports better decision-making for students, faculty, and administrators.

TECH STACK
  • React
  • Node.js
  • MySql
  • Tailwind

Library Management System

A desktop-based Library Management System developed using VB.NET, designed to streamline and centralize the organization and tracking of library resources. The system integrates book catalog management, borrowing and return processes, user account handling, and real-time inventory monitoring into a single, unified application. By providing efficient access to book records, automated due date tracking, and a user-friendly interface, the system enhances operational efficiency, improves data accuracy, ensures accountability, and supports a smoother experience for both librarians and users.

TECH STACK
  • Visual Basic (VB)
  • Oracle
